# BAGEL-GAME
The the game begins with one person secretly choosing a number with no repeated digits. We will start with a three digit number. Then others attempt to guess the number, and the one who chose the number responds to each guess as follows: • If the guess has no numbers correct, they respond: "Bagel." • For each digit the guess has correct, but in the wrong place, they respond: "Pico." • For each digit the guess has correct and in the correct place, they respond: "Fermi."
